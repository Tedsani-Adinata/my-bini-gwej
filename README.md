# Character Gallery

A static character gallery built with HTML, CSS, and JavaScript. Each panel contains a video preview that plays on hover, with character information revealed inside the active panel.

## Project Structure

```text
.
|-- index.html
|-- styles.css
|-- script.js
`-- images/
    |-- kei.mp4
    |-- nagusa.mp4
    |-- nagisa.mp4
    |-- hina.mp4
    |-- suzumi.mp4
    |-- toki.mp4
    `-- Kei_01.png
```

## Preview

Open `index.html` in a browser, or run the folder with a local server such as the VS Code Live Server extension.

## Editing

- Update the page structure and character text in `index.html`.
- Adjust layout, animation, spacing, and colors in `styles.css`.
- Change hover playback behavior in `script.js`.
- Replace videos or images inside the `images` folder.

When replacing media files, keep the file names the same or update the matching paths in `index.html`.
